# ❄️ 線上色卡管理系統

這是一款專為設計師與開發者打造的輕量化色彩管理工具。採用 **Winter Blossom（冬日花卉）** 美學主題，將 Excel 配色表轉化為視覺化的數位色卡牆。

## ✨ 核心特色
- **自動同步**：預設自動讀取同目錄下的 `color.xlsx`。
- **強韌解析**：智慧過濾儲存格雜質，精準提取 `#RRGGBB` 色碼。
- **美學互動**：
  - **冬花粉 (#F8B5B5)**：精緻的懸停與點擊回饋效果。
  - **智慧對比**：色塊文字根據背景深淺自動切換黑/白。
- **響應式佈局**：針對手機與寬螢幕優化，自動填滿橫向空間，絕不留白。
- **一鍵複製**：點擊色塊或按鈕即可快速獲取色標。
