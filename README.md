# Color Palette Reader

這是一個基於瀏覽器的色彩管理工具，採用「冬日花卉 (Winter Blossom)」美學設計。
只需上傳 Excel 檔案，即可自動解析多個工作表，生成視覺化的色卡牆。

## ✨ 功能特色
- **Excel 全解析**：支援 SheetJS，自動讀取多個分頁。
- **智慧辨識**：自動抓取欄位中的 HEX 色碼 (#RRGGBB)。
- **繁體校正**：內建文字處理，自動將「栌」校正為「櫨」。
- **一鍵複製**：點擊色塊即可複製代碼。

## 📂 如何使用
1. 下載範例檔案：[color.xlsx](./color.xlsx)
2. 將檔案拖曳至網頁上傳區。
3. 開始探索你的配色資料庫！
